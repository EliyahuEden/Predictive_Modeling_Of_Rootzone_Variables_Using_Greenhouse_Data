Model workflow

Strict walk-forward validation - no data leakage, no look-ahead. The model is retrained after each new labeled measurement is revealed.

Training & prediction pipeline
1
Warmup phase
Train initial model on first 50 pH intervals (first 40 for EC). No predictions made during warmup.
2
Predict at t1
At each new measurement timestamp, extract features from [t0->t1) window + 24h pre-t1 history. Run model to predict pH/EC change.
3
Reveal & retrain
True value is revealed. Add this interval to training set and retrain model. Training set grows by 1 after each sample.
4
Repeat
Steps 2-3 repeat for every subsequent labeled timestamp. Final training set = 109 samples (all data).
