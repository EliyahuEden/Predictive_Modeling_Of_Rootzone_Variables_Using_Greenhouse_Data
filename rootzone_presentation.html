Model overview

Two XGBoost models predict pH and EC at the next rootzone sample. V5 adds 24h pre-t1 history features to V3's window-only approach — all 7 history features anchored to t1.

What is V3?
V3
Window-only features (t0 → t1)

22 pH features, 14 EC features. All information comes from the interval being predicted: cumulative irrigation, fertilization chemistry, climate variables, interaction terms.

What did V5 add?
V5
+ 7 history features from 24h before t1

Sub-windowed irrigation history (0–6h, 6–12h, 12–24h before t1), exponential-decay acid dose, net salt buildup, hours-since-irrigation and hours-since-fertigation — all anchored to t1. Captures biological lag: nitrification peaks at 8–16h, pH buffering takes 12–24h.

~4 months of 10-minute greenhouse logs with sparse manual rootzone measurements.

Model workflow

Strict walk-forward validation — no data leakage, no look-ahead.

Training & prediction pipeline
1
Warmup

First 50 intervals (pH) / 40 intervals (EC) train the initial model. No evaluation during warmup.

2
Feature extraction at each t0

Extract window features from [t0, t1). V5 additionally reads the 24h block before t1 for history features — all anchored to t1.

3
Predict Δ, evaluate at t1

Model predicts the change (delta) from t0. Final prediction = ph0 + Δph. Error recorded vs real measurement.

4
Retrain on all data seen so far

After each real label is revealed, the model retrains on the full growing training set. Improves continuously.

5
Baseline comparison

Every prediction compared to the naive baseline (carry forward last known value). Model must beat naive to justify complexity.

No-leakage guarantees
WhatRule
Window featuresUse [t0, t1) — t1 event excluded
History features (V5)24h before t1 — all anchored to t1
ValidationAlways on future unseen intervals
Retrain timingOnly after real label revealed
TargetPredict Δ not absolute value
Why predict Δ instead of absolute value?
Predicting the change from t0 anchors every prediction to a known real measurement. This eliminates systematic drift and makes the model invariant to seasonal trends in absolute pH/EC levels.
