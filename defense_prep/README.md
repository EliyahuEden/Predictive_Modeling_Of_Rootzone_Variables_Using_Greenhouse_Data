# Rootzone Poster Defense Kit

This folder contains the practical preparation materials for the 30-minute poster defense.

Use these files in this order:

1. `01_timed_speaker_script_he.md` - rehearse the full 22-minute talk.
2. `02_partner_alignment_story_he.md` - align Eden and Oren on the project chronology and decision story.
3. `03_judge_qa_bank_he.md` - practice likely judge questions.
4. `04_dashboard_demo_runbook_he.md` - prepare the live dashboard demo and fallback.
5. `05_metric_cheat_sheet_he.md` - memorize the final metrics and definitions.
6. `06_printable_backup_handout_he.md` - print as the one-page backup sheet.
7. `07_rehearsal_checklist_he.md` - use before each rehearsal and on presentation day.
8. `defense_control_room.html` - open in a browser during rehearsal for a compact timer-style view.

- `screenshots/dashboard_home.png` - dashboard landing screenshot.
